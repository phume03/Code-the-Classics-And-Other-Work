# Code-the-Classics And Other-Work From RaspberryPiPress #

Raspberry Pi Press is the book publishing division of Raspberry Pi Holdings, established in 2015. It produces magazines and books related to the Raspberry Pi computer. One such production is the book series, "Code the Classics", available for sale on Amazon but also seemingly made free by RaspberryPi Press here on Github. The book authors are David Crookes, Andrew Gillett, Liz Upton, and Eben Upton. It was published on 13 Dec 2019. According to a description from [volume 1](https://magazine.raspberrypi.com/books/code-the-classics-vol-I-2ed) of the two volume book series, "Code the Classics" not only tells the stories of some of the seminal video games of the 1970s and 1980s, but shows you how to create your own games inspired by them, following examples programmed by Raspberry Pi founder Eben Upton. The books apparently teach you how to run and edit the games that they present. The primary tools are Python, Pygame Zero, and an IDE (Github Co-pilot is data heavy). 

You want them because they will give you:

+ Get game design tips and tricks from the masters
+ Learn how to code your own games with Pygame Zero
+ Explore the code listings and find out how they work
+ You'll meet vintage-inspired games, and learn from their code in between rounds of play



Assets from the Code the Classics books:

+ Boing!: all it took was a couple of lines and a dot, and gamers would be queuing up to play.
+ Cavern: Enduringly popular, the platform game genre is still packed with creative possibilities.
+ Infinite Bunner: Play around with the benefits that a top-down perspective can lend to the classic platform genre.
+ Myriapod: Some shooters confine the gameplay to a single screen while limiting the player's movement. Restrictions can build challenge and difficulty, making for truly addictive gaming.
+ Substitute Soccer: Top-down games of pinball-style soccer built a huge cult following and kicked off a sports genre that's still going strong.

+ Kinetix: An Arkanoid-style block breaker game​
+ Avenger: A horizontal 2D shooter game like Defender​
+ Eggzy: An action-adventure tribute to Dizzy​
+ Beat Streets: A take on the Beat-'em-up classic Double Dragon​
+ Leading Edge: An arcade racer homage to Pole Position

## Other work ##

[Make Games with Python](https://www.raspberrypi.com/news/make-games-with-python-2nd-edition-out-now/) was authored by Sean M. Tracey (Author), Russell Barnes (Editor). I found this under the RaspberryPi Press Github repo. It seems like a perfect companion to the Code the Classics collection. An online book description says that it is the 2nd book in The MagPi Essentials range, and is designed to help you learn the coding skills you need to create amazing games and applications. The focus is the raspberry pi and i assume that is why code source and books were made available free online to promote the platform. It is true that millions of us like enjoy spending hours racking up high scores on our favourite games, and even more true that too few of us are exposed to an even more gratifying way to spend the evening - making them!!! 

So I am picking up this book to master essential game-making skills across 10 chapters Creating shapes and paths; Movement and animation; Using the keyboard & mouse; Adding sound and music; Simulating physics and forces; Building classes for actors; Creating your own shoot-em 'up.
